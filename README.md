- make the script executable

```bash
chmod +x compile.sh 
```

- run the script

```bash
./compile.sh 
```

## Options

- `-l <length>`: Specify the password length (default: 8)
- `-u`: Include uppercase letters
- `-n`: Include numbers
- `-s`: Include symbols
- `-e`: Generate an easy-to-remember password
- `-v`: Show version
- `-h`, `--help`: Display help message

**Example:**

```bash
pass 12 -u -n
```
